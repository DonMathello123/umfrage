# Abi-25-Web-App – Online-Umfrage per QR-Code

Dieses Paket enthält eine kleine Smartphone-Web-App für das 25-jährige Abitreffen.

## Dateien

- `index.html` – die eigentliche Umfrage-App
- `google_apps_script_backend.gs` – Backend zum Speichern in Google Sheets
- `ANLEITUNG.txt` – Schritt-für-Schritt-Anleitung

## Wichtig

Die App funktioniert optisch sofort, aber für das zentrale Sammeln aller Antworten brauchst du ein Google-Sheet-Backend.

Ohne Backend werden Antworten nur lokal im Browser des Teilnehmers gespeichert.
